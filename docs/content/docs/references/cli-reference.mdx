## Core Commands

### `turboversion` - Version Management
By default `turboversion` runs automatic versioning based on commit messages or branch names. Use `--bump` to specify version types directly.

```bash
turboversion --bump [options]
```
**Options**:
| Flag | Description | Example |
|------|-------------|---------|
| `--bump major`/`--minor`/`--patch` | Force version bump type | `--bump major` |
| `-p`, `--prerelease` | Create prerelease using auto-detected bump type | `--prerelease` |
| `--target <package name>` | Target specific package | ` --target @app/core` |
| `--async` | Async mode (monorepos) | `--async` |
| `--sync` | Sync mode | `--sync` |

**Examples**:
```bash
# Minor release with changelog
turboversion --bump minor

# Prerelease with auto-detected bump type (feat -> preminor, fix -> prepatch)
turboversion --prerelease

# Explicit prerelease type
turboversion --bump prerelease

# Monorepo selective update
turboversion --bump minor --target @ui/button
```

### `init` - Configuration Setup
```bash
turboversion init [options]
```
**Options**:
| Flag | Description | Example |
|------|-------------|---------|
| `--force` | Overwrite existing config | `init --force` |
| `--defaults` | Skip prompts | `init --defaults` |

## Global Flags
Available for all commands:

| Flag | Description | Example |
|------|-------------|---------|
| `--config <path>` | Custom config file | `--config ./custom.config.json` |
| `--version` | Show version | `--version` |
| `--help` | Show help | `--help` |

## Common Workflows

### 1. Production Release
```bash
turboversion --bump minor && git push --follow-tags
```

### 2. Monorepo Hotfix
```bash
turboversion --bump patch --target @lib/utils
```

### 3. CI/CD Integration
```bash
turboversion --bump minor
```

### 4. Prerelease with auto-Detection
```bash
# Detects bump type and creates prerelease
# feat commit/branch -> preminor (1.0.0 -> 1.1.0-beta.0)
# fix commit/branch -> prepatch (1.0.0 -> 1.0.1-beta.0)
turboversion --prerelease
```

|------|---------|-----------------|
| 0 | Success | - |
| 1 | General error | Check debug logs |
| 2 | Configuration error | Validate config file |
| 3 | Version conflict | Resolve git state |
| 4 | No changes detected | Verify commit history |
